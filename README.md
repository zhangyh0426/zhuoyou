# 狼人杀桌游店微信小程序

这是一个专为狼人杀桌游店设计的微信小程序，具有以下功能：

## 功能特性

1. **会员系统**
   - 用户登录/注册
   - 会员卡余额管理
   - 消费记录查询

2. **店铺状态**
   - 实时显示店铺是否营业
   - 营业时间设置（仅管理员）
   - 开业/打烊状态切换（仅管理员）

3. **活动管理**
   - 创建狼人杀活动
   - 查看所有活动列表
   - 报名参加活动
   - 实时显示报名人数

## 使用说明

1. 在微信开发者工具中导入项目
2. 编译并运行项目
3. 使用管理员功能需要在代码中设置特定手机号为管理员

### 管理员功能

要使用管理员功能（修改店铺状态、设置营业时间），请按以下步骤操作：
1. 打开 `app.js` 文件
2. 找到 `initDefaultAdmin()` 方法中的超级管理员配置
3. 默认超级管理员手机号为：13614470578
4. 您可以根据需要修改管理员信息或添加更多管理员

示例配置在 `app.js` 文件中：
```javascript
initDefaultAdmin() {
  const defaultAdmins = [
    {
      phone: '13614470578',  // 超级管理员手机号
      name: '店长',
      role: 'super'  // super 表示超级管理员
    },
    {
      phone: '13900000000',  // 普通管理员手机号
      name: '副店长', 
      role: 'normal'  // normal 表示普通管理员
    }
  ];
  // ...
}
```

## 数据存储

本小程序使用微信小程序的本地存储功能来保存数据：
- 会员信息
- 店铺状态
- 活动信息
- 报名记录

## 注意事项

1. 这是一个单机版小程序，所有数据都存储在本地
2. 如需多人共享数据，需要连接后端服务器
3. 清除小程序缓存会丢失所有数据

## 页面说明

### 首页 (/pages/index/index)
- 显示欢迎信息
- 展示店铺当前营业状态
- 提供快捷操作入口（创建活动、会员卡）
- 显示今日活动和推荐活动

### 登录页 (/pages/login/login)
- 手机号和姓名登录/注册
- 提供演示账号快速登录

### 会员卡页 (/pages/member/member)
- 显示会员个人信息
- 查看账户余额
- 充值功能
- 消费记录查询

### 店铺状态页 (/pages/store-status/store-status)
- 显示店铺详细营业状态
- 管理员可切换开业/打烊状态
- 管理员可设置营业时间

### 活动创建页 (/pages/activity/activity)
- 创建新的狼人杀活动
- 设置活动标题、日期、时间、人数限制等

### 活动列表页 (/pages/activity-list/activity-list)
- 查看所有活动
- 按今日/近期活动筛选
- 报名参加活动
- 查看活动详情
